Definition

The View is the user interface of the application. It displays information received from the Model through the Controller.

Responsibilities

  • Displays data
  • Provides user interface
  • Receives user input
  • Shows updated information
  • Communicates with Controller

User Interface Elements

  • Buttons
  • Forms
  • Text Boxes
  • Tables
  • Images
  • Menus
  • Labels
