Definition

The Model represents the application's data and business logic. It is responsible for storing, retrieving, and updating information.

Responsibilities

  • Stores application data
  • Processes business logic
  • Updates database records
  • Validates data
  • Provides data to the Controller

Examples of Data

  1. Student Details
  2. Employee Records
  3. User Login Information
  4. Product Details
  5. Bank Account Information
